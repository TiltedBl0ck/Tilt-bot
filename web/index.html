A Feature for Every Need

Discover the powerful tools Tilt-bot puts at your fingertips.

Gemini AI Chat

Engage your community with intelligent, context-aware conversations powered by Google's Gemini API. Use `/chat` to talk or mention the bot.

Advanced Setup & Config

Use intuitive modals and dropdowns with `/setup` and `/config` to customize welcome messages, goodbye alerts, and server stats effortlessly.

Full Moderation Suite

Keep your community safe with a complete set of moderation commands, including kick, ban, timeout, and message clearing, all with hierarchy checks.

Easy to Use Commands

All commands use Discord's built-in slash commands. Just type `/` to get started.

Tilt-bot features a dynamic and interactive help command. To see a full, up-to-date list of every command and what it does, simply use the following command in your server:

/help
