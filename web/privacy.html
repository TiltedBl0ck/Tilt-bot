Privacy Policy

For the Tilt-bot Discord Bot

Last updated: September 29, 2025

1. Information We Collect

Tilt-bot is designed to be privacy-focused. We only store data that is essential for its features to function correctly. This includes:

  • Server Configurations: We store server-specific settings, such as the designated channels for welcome/goodbye messages and server statistics. This data is associated with the Server ID (Guild ID).
  • User-Specific Data (AI Chat): To provide a continuous conversation experience with the `/chat` command, we temporarily store a history of your conversation with the Gemini AI. This history is associated with your User ID.
  • Command Usage (No User Data): The bot does not store the content of messages, except for prompts sent directly to the AI features. We do not log user messages in channels.

We do **not** collect or store personal information like email addresses, usernames, or user messages (outside of AI prompts).

2. How We Use Information

The data collected is used exclusively for the following purposes:

  • To provide and operate the Bot's features, such as sending welcome messages to the correct channel.
  • To maintain conversation context for the AI chat feature.
  • To ensure the stability and security of the Service.

3. Data Storage and Security

All stored data is kept on a secure, private database. We take reasonable measures to protect the information we store, but no security system is impenetrable.

4. Data Sharing

We do not share, sell, rent, or trade any collected data with third parties for any purpose.

5. Data Retention and Deletion

Server configuration data is retained as long as the bot remains in the server. When the bot is removed from a server, the associated configuration data may be deleted. You can request the deletion of your server's data at any time by contacting the bot developer.

6. Contact Us

If you have any questions about this Privacy Policy, please contact the developer, TiltedBl0ck.
